========
halo5.py
========

A wrapper for the `Halo 5: Guardians API`_.

.. _`Halo 5: Guardians API`: https://developer.haloapi.com/

Installation
------------

To install ``halo5.py`` ::

  pip install halo5.py


Tests
-----

To run the tests, run the following script ::

  ./test "my-halo-5-api-subscription-key"
